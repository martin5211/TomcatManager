# Privacy Policy

## Data Collection

Tomcat Manager for VS Code does **not** collect, transmit, or store any personal data or telemetry.

## Configuration

All configuration (servers, projects, JVM options) is stored locally in your VS Code settings. No data is sent to external servers.

## Permissions

The extension accesses only:

- **Local file system** — to deploy WAR files and manage Tomcat directories (`webapps/`, `work/`, `temp/`)
- **Local processes** — to start and stop Tomcat instances on your machine
- **VS Code settings** — to read your `tomcatManager.servers` and `tomcatManager.projects` configuration

## Third-Party Services

This extension does not communicate with any third-party services or APIs.

## Contact

If you have questions about this policy, open an issue at [https://github.com/martin5211/TomcatManager](https://github.com/martin5211/TomcatManager).
